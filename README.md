# Consulta de CEP com Python

Projeto simples para consumo de API pública utilizando Python.

## Tecnologias
- customtkinter 
- Python
- Requests
- API ViaCEP

## Objetivo
Consumir uma API pública, tratar os dados retornados e exibir informações organizadas no terminal.

## Como executar
1. Instale as dependências:
pip install -r requirements.txt

2. Execute o script:
python app.py

3. Digite o CEP desejado.
